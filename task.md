# 项目任务清单 (Task List)

## 项目整体描述
`tabby-quick-scripts` 是一个为 Tabby 终端定制的快捷脚本插件。该插件可以在 SSH 等终端界面的顶部注入一条快捷操作栏，允许用户一键按顺序执行多行命令、实时监听命令提示符进行等待，并支持在界面上对快捷脚本进行创建、编辑、删除等可视化管理。

## 开发规范描述
1. **KISS 原则**：保持简单，避免过度设计，注重可维护性。
2. **语言及框架**：基于 Angular 12、TypeScript、Webpack 5 构建，遵循 Tabby 插件开发规范。
3. **构建流程**：所有对 `src/` 目录的改动必须经由 `npm run build` 或 `npm run watch` 编译至 `dist/` 目录才能生效。
4. **代码风格**：遵循项目既有的缩进及命名习惯（Tabby 的风格）。
